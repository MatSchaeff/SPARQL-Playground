LEARN

For students
learning
SPARQL
User-friendly environment
The SPARQL playground is a user-friendly environment for newbies to start learning SPARQL.
A simple scenario with less than 100 triples is configured by default with diagrams.
The basics of SPARQL are explained with comprehensive examples.
Online or offline
Either access the online version at:
  • https://sparql-playground.isb-sib.ch
    • or download it locally hence benefiting from all features.
    For example the RDF dataset can be edited and reloaded on the fly in the localhost version.

    TEACH

    For instructors
    teaching
    SPARQL
    Platform-independent
    The application can be downloaded and installed in any student's laptop.
    It requires less than 30MB of hard disk and works on any operating system, as long as Java is installed.
    These inherit characteristics makes the SPARQL playground a good candidate to teach SPARQL on tutorials and workshops.
    No network and concurrency issues
    Since the application is installed locally on students' machine, there are no issues about setting up internet connection or relying on a single remote endpoint.
    For the same reason, problems related to concurrent requests at a single endpoint don't exist anymore.

    DEVELOP

    For developers
    building a
    datamodel
    Dev / Test environment
    The application comes with a default scenario, but can be configured to work with any RDF data and example queries. This playground can be built for users to get to know your model with nice examples, before hitting the real data
    Technology (Sesame Engine)
    The backend is developed with SesameDB SPARQL engine + Java Spring Boot and the frontend with AngularJS + Bootstrap. All the source are open source and contribution is welcome.

    BUILD YOUR PLAYGROUND WITH YOUR DATA

    No programmatic skills are required to set a custom environment
    The only requirement: Java 1.7 or higher

    1. Download the latest stable release and extract it :
      https://github.com/calipho-sib/sparql-playground/releases
    2. Create a directory according to the figure below ex: "my-data"
    3. Edit the prefixes.ttl file in SPARQL format
    4. Add all RDF data in turtle format inside the ttl-data folder
    5. Put all example queries inside the queries folder
    6. Start the java script :
      java -jar sparq-playground.war my-data
